# Quran Hifz App

This is a Quran memorisation (Hifz) learning application.

## Purpose
To help students track, revise, and strengthen Quran memorisation in a structured way.

## Features (planned / in progress)
- Hifz progress tracking
- Revision system
- Structured learning folders (lib, scripts, assets)
- Interactive learning tools

## Project structure
- lib/ → core logic
- scripts/ → helper scripts
- attached_assets/ → images, audio, resources
- artifacts/ → generated or saved outputs

Early development stage
